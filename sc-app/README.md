# 东软智慧社区 - 用户端

Vue3 + Vite + Element-Plus + Pinia 用户端前端（商城 + 社区服务）。

## 开发

```bash
npm install
npm run dev
```

默认访问 http://localhost:5174 ，通过 Vite 代理将 `/api` 转发到后端网关 http://localhost:8080。需先启动网关及各微服务（sc-auth、sc-user、sc-mall、sc-trade、sc-community、sc-search、Nacos）。

## 构建

```bash
npm run build
```

产物在 `dist/`，可部署到静态服务器并配置 `/api` 反向代理到网关。

## 功能概览

- **登录/注册**：手机号+密码，Token 存本地，请求头带 Authorization。
- **商城**：首页（分类、门店列表）、搜索商品、门店详情（商品列表、加购）、商品详情（可选门店加购）、购物车、确认订单、我的订单（支付/取消/确认收货）。
- **社区**：公告列表、访客登记、我的车辆（绑定/解绑）、报修（提交/我的/取消）、投诉建议（提交/我的）、物业费（我的账单/缴纳）、车位查询（按小区ID）。
- **我的**：个人资料与钱包余额、修改昵称、修改密码、购物车与订单入口、退出登录。

底部导航：商城 | 社区 | 我的。需登录的页面未登录会跳转登录页。
