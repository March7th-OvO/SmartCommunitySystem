# 东软智慧社区 - 管理端

Vue3 + Vite + Element-Plus + Pinia 管理端前端。

## 开发

```bash
npm install
npm run dev
```

默认访问 http://localhost:5173 ，通过 Vite 代理将 `/api` 转发到后端网关 http://localhost:8080 ，需先启动网关及各微服务。

## 构建

```bash
npm run build
```

产物在 `dist/`，可部署到任意静态服务器，并配置反向代理将 `/api` 指向网关。

## 功能

- **登录**：手机号 + 密码，对接 `POST /api/auth/login`，登录后 Token 写入本地并带在请求头。
- **工作台**：快捷入口。
- **用户管理**：用户列表（分页、手机号/状态筛选）、启用/禁用、分配角色。
- **商城管理**：分类（树形增删改）、门店（分页增删改）、商品（分页增删改）。
- **订单**：管理端订单分页、发货、确认完成、订单明细。
- **社区物业**：公告、访客审核、报修处理、投诉处理、物业费账单（新建+分页）、车位列表（按小区ID查询）。

路由守卫：未登录访问需登录页会跳转登录；登录后访问 `/login` 会跳转首页。
